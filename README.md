# SFMShop

Backend интернет-магазина, разработанный на Python и FastAPI.

Проект создан для изучения современных технологий backend-разработки, включая REST API, работу с базами данных, кэширование, асинхронное программирование и контейнеризацию.

## Возможности

- Управление товарами
- Управление пользователями
- Создание заказов
- Кэширование через Redis
- Ограничение количества запросов (Rate Limiting)
- Асинхронная обработка операций
- Логирование действий приложения
- Автоматические тесты
- Миграции базы данных через Alembic

## Технологии

- Python 3
- FastAPI
- PostgreSQL
- SQLAlchemy
- Redis
- Docker
- Alembic
- Pytest
- Asyncio

## Запуск проекта

### Клонирование репозитория

git clone <repository_url>

cd SFMShop

### Запуск через Docker

docker-compose up --build

### Запуск тестов

pytest
